# Quiz Application (Python)

## Project Description

The Quiz Application is a menu-less, interactive Python program that presents multiple-choice questions to the user and evaluates their answers.
This project is built using basic Python concepts and helps beginners understand logic building, data handling, and conditional evaluation.

The application calculates the final score, percentage, and displays a PASS or FAIL result based on performance.

---

## Features

* Multiple-choice quiz system
* Predefined set of questions
* Automatic score calculation
* Percentage-based result evaluation
* PASS / FAIL result display
* Simple and interactive terminal output

## How to Run the Program

### Requirements

* Python 3.x installed on the system

### Steps

1. Open terminal or command prompt
2. Navigate to the project directory
3. Run the following command:

```
python quiz.py
```

---

## Quiz Flow

```
1. Question is displayed
2. Four options (A/B/C/D) are shown
3. User selects an option
4. Score is updated based on correctness
5. Final result is calculated
```

---

## Result Criteria

* Percentage ≥ 55 : PASS
* Percentage < 55 : FAIL
